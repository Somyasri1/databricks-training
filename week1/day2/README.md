#  Databricks SQL Training — Day 2

##  Overview

This directory contains Day 2 SQL practice exercises completed as part of Databricks SQL Training. The focus of this session is on advanced SQL concepts such as joins, subqueries, aggregate analysis, nested queries, and data analysis techniques used in real-world business scenarios.
